About Me

I attended Nashville Software School with Cohort 11, the best Cohort because we are #1 twice. It is a 40+ hours per week, intensive web development bootcamp. The first three months focus on front end development. During this time we designed interactive static and dynamic websites. While we began with the basics, using only Javascript, CSS, and HTML, we began to use other tools to build out larger, more complex websites and applications.

The second three months focus on back end developmend via the Node MEAN stack (Mongo, Express, Angular, and Node). We also covered SQL databases, including sqlite, PostgreSQL, and the Sequalize ORM tools.
